# React Learning Projects

A collection of React projects built while learning and practicing React concepts.

## Overview

This repository contains multiple mini-projects and demonstrations created to understand and master various React fundamentals and advanced concepts.

## Projects

### 1. **Catalogue Page Demo**
A React application demonstrating a product catalogue/listing page.
- Location: [`catalogue-page-demo/`](catalogue-page-demo)
- Features: Component-based UI, catalogue display
- Tech: React, CSS

### 2. **Pure React**
Vanilla React implementation exploring core React concepts.
- Location: [`pure_react/`](pure_react)
- Features: Basic React patterns without additional frameworks

### 3. **JavaScript Experiments**
Raw JavaScript implementations and experiments related to React concepts.
- Location: [`js/`](js)
- Features: Pure JavaScript examples

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Running Projects

#### Catalogue Page Demo
```bash
cd catalogue-page-demo
npm install
npm start
```

## Learning Goals

This repository is a learning journey covering:
- ✅ React fundamentals (components, JSX, props, state)
- ✅ Component lifecycle and hooks
- ✅ Styling and CSS in React
- ✅ State management
- ✅ Building reusable components
